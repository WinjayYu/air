## Introduction

This is a simple single-page for Broccoli & Co. It based on vue 

## Usage

**Step 1**. Clone this repository

```shell
$ git clone https://github.com/WinjayYu/air.git
$ cd air
$ npm install
```

**Step 2**. development

```shell
$ npm run dev
```
then open index.html in the browser

**Step 3**. build

```shell
$ npm run build
```
then open index.html in the browser

**Step 4**. test

```shell
$ npm run test
```
